Ce projet est une application de gestion de tâches (todolist) permettant aux utilisateurs de créer et organiser leurs listes de tâches de manière intuitive. L'objectif est de fournir une solution moderne et performante pour la gestion des tâches quotidiennes, en utilisant React.js pour l'interface utilisateur et GraphQL pour la gestion des requêtes côté serveur.
Commandes de compilation: 
	npm install : pour télécharger le dossier node_modules
	npm start : pour lancer l'application
